# Fixtures

**Two categories only:**

1. **Public-domain captures** sourced from Internet Archive (with the URL
   recorded in this directory's `provenance.json`).
2. **Synthetic signals** generated by `SyntheticIq` and friends, marked with
   `ProvenanceTag::synthetic(...)`.

**Forbidden by CI** (see `crates/ethics-audit/`):

- Broadcast recordings of any kind (RDS / SCTE-35 / callsign-bearing).
- Recordings made in colonial contexts without community-of-origin
  acknowledgement metadata.
- Any capture lacking a recording-year ISO date.

The ethics audit runs over this directory at every PR. A fixture that fails
audit blocks the PR; it is not "warned about."

## v0.1 contents

Empty by design — v0.1 generates synthetic signals at test time. v0.2 will
add a `scripts/fixtures-fetch.sh` that downloads from Internet Archive into
a `fixtures/internet-archive/` subtree with a recorded `provenance.json`.
